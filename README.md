# Projet C : Super-Heros
Dans le cadre du module de langage C, ce mini-projet a pour objectif principal de permettre à un utilisateur de naviguer parmi les informations de différents super-héros présentes dans un fichier JSON, de voir leurs statistiques détaillées ainsi que de comparer leurs attributs.
Le programme créé propose des interactions avancées telles que des recherches avancées, une gestion de filtres modulables, un quizz ainsi qu’un système de gestion de favoris. 

## structure du dépot :
    /code contient le fichier main.c qui est le code principal ainsi que l’executable ProjetC.exe qui est le programme fonctionnel
    /data contient les données SuperHeros.json fav.json ainsi que scores.json
    /cJSON contient fichiers .c et .h de la bibliothèque cJSON
    Pour le bon fonctionnement du programme il ne faut pas deplacer les fichiers et veiller à ce que la totalité des fichiers cités ci-dessus soient présent à la bonne place
