# Projet C : Super-Heros

## structure du dépot :
    /code contient le fichier main.c qui est le code principal ainsi que l’executable ProjetC.exe qui est le programme fonctionnel
    /data contient les données SuperHeros.json fav.json ainsi que scores.json
    /cJSON contient fichiers .c et .h de la bibliothèque cJSON
    Pour le bon fonctionnement du programme il ne faut pas deplacer les fichiers et veiller à ce que la totalité des fichiers cités ci-dessus soient présent à la bonne place
